# Text Editor
You are planning to implement the Text editor as a GUI.  
Using Python3 and tkinter, etc...

## Enable Functions  
* New
* Open
* Save

very simple  

## Features
* variable font size
* support to more font
* support user setting
* enable to variable window size
* change to program icon

* python tkinter

```bash 
pip3 install python-tk
```

python 3.x
